报道所援引的消息源来自网友在飞行航迹网站上的发现。报道称，有网友发现，一架没有识别码，显示“nosign”的疑似中共侦察机，在上午11时27分，飞行到台湾西南空域时，接着在附近开始盘旋，来回的绕圈圈，这一绕，绕到下午14时20分，这架疑似共机至少在台海周边绕圈至少173分钟。

“三立新闻网”还将尚未得到证实的解放军军机行动渲染为“挑衅”。报道声称，“挑衅”的时机敏感，因为蔡英文于23日搭乘专机前往金门太武山公墓主持阵公亡将士追思活动，而共机“恰巧”就在专机返程时，出现在台湾海域附近。

对于解放军军机频频在台海活动一事，东部战区新闻发言人张春晖空军大校曾就海空兵力战备巡航发表谈话，他强调，台湾及其附属岛屿是中国领土神圣不可分割的一部分，中国军队战备巡航完全正当合法，是针对当前台海安全角势和维护国家主权需要采取的必要行动。战区部队有决心、有能力挫败一切“台独”分裂活动，坚决捍卫国家主权安全和领土完整，坚决维护台海地区和平稳定。
